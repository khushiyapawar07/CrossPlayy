# 🎮 Gaming Café Slot Booking System

A full-stack web application designed to manage and streamline slot bookings for a gaming café. The system allows users to reserve gaming stations (PS5 or PC) in real-time, while providing admins with tools to manage stations, bookings, and pricing.

---

## 🚀 Features

### 👤 User

* Register & Login authentication
* View available slots (PS5 / PC)
* Real-time slot booking
* Booking history
* Cancel reservations

### 🛠️ Admin

* Manage gaming stations (add/remove/update)
* View all bookings
* Control slot availability
* Monitor usage

---

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/khushiyapawar07/CrossPlayy.git

```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
